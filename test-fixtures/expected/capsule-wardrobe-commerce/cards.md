# 图文笔记内容

## 页面结构

### 封面页
- 页型：封面页
- 页面目标：纠正胶囊衣橱的误解：它不是单纯少买，而是降低搭配决策成本。
- 入图文字：
  - 大标题：胶囊衣橱不是少买衣服
  - 副标题：而是让搭配变简单
  - 标签：基础底色 / 质感外层 / 少量记忆点
  - 标语：让每一件衣服更好搭
- 语义映射：用法式衣橱静物和三类单品标签表达搭配系统。
- 避免误读：不要做成极简断舍离鸡汤，也不要做成密集商品目录。

### 内容页1/3
- 页型：对比页
- 页面目标：区分“少买衣服”和“更好组合”的差别。
- 入图文字：
  - 大标题：核心不是越少越好
  - 误解：胶囊衣橱 = 少买
  - 正确：让单品更容易互相搭配
  - 关键：组合效率
  - 总结：减少的是决策成本
- 语义映射：左右双栏，左侧是稀少但孤立的单品，右侧是可互搭的衣橱组合。
- 避免误读：不要把主题变成消费克制说教。

### 内容页2/3
- 页型：框架页
- 页面目标：讲清胶囊衣橱的三类核心单品。
- 入图文字：
  - 大标题：好用衣橱的 3 类单品
  - 基础底色：决定可组合程度
  - 质感外层：让搭配更完整
  - 少量记忆点：形成个人风格
  - 总结：少量重点，比盲目追新更有用
- 语义映射：三格精品分类网格，每格一类单品和一个静物视觉。
- 避免误读：不要列过多商品，不要像电商货架。

### 内容页3/3
- 页型：清单页
- 页面目标：给出购买前的三个判断问题。
- 入图文字：
  - 大标题：买之前先问 3 个问题
  - 能和已有 3 件单品搭配吗？
  - 适合我的真实生活场景吗？
  - 颜色和材质能融入现有衣橱吗？
  - 总结：值得买 = 能让更多搭配成立
- 语义映射：优雅购物清单/衣橱检查卡，三条问题是主信息。
- 避免误读：不要生成价格、品牌或促销信息。

## 发布配文

胶囊衣橱不是让你少买，而是让每一件衣服都更好搭。

真正好用的衣橱，通常离不开三类单品：基础底色、质感外层、少量记忆点。基础款决定组合度，外层决定完整度，记忆点决定个人风格。

买之前可以先问：能不能和已有 3 件单品搭配？适不适合真实生活场景？颜色和材质能不能融入现有衣橱？

一句话总结：胶囊衣橱不是限制选择，而是降低每天搭配的决策成本。
