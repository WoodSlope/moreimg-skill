# 文生图提示词

## 使用风格

xhs-french-editorial-commerce｜小红书法式电商杂志风

## Style Lock

法式电商杂志风，奶油白和焦糖棕主导，低饱和暖调，真实感时装摄影或产品静物，规则图片网格或大图留白结构，粗窄体大标题，优雅衬线小字，细线分割，轻微胶片颗粒，留白充足，中文短文案可读。中文优先，保留必要英文术语或风格标签。统一使用米色细线、无边框图片网格、细边框标签、低矮页脚、衬线说明。不要混入科技 HUD、银灰蓝知识卡、机构报告黑线网格、赛博朋克、黑金奢华或卡通插画。

## 提示词

### 封面页

[3:4, 1K] 生成一张小红书图文卡片，页型是「封面页」，主题是「胶囊衣橱搭配逻辑」。
画面气质：法式电商杂志风，奶油白与焦糖棕主导，低饱和暖调，真实感衣橱静物和布料细节，轻微胶片颗粒，优雅留白。
排版：上方强标题区，中部一组法式衣橱静物和三类单品标签，底部短标语，细线分割，边距稳定，手机端可读。
内容层级：大标题「胶囊衣橱不是少买衣服」，副标题「而是让搭配变简单」，要点包括「基础底色」「质感外层」「少量记忆点」，总结「让每一件衣服更好搭」。
视觉锚点：奶油色背景上的衣架、白衬衫、风衣、丝巾、皮革腰带等静物，组成克制的衣橱系统。
色彩：奶油白、焦糖棕、深巧克力文字，少量酒红或橄榄绿点缀，柔和棚拍光影。
避免：极简断舍离说教、密集商品目录、促销标签、黑色外框、霓虹科技、随机英文假字。

### 内容页1/3

[3:4, 1K] 生成一张小红书图文卡片，页型是「对比页」，主题是「胶囊衣橱的常见误解」。
画面气质：法式电商杂志风，暖调纸感，优雅衬线小字，焦糖棕细线，真实感衣物静物，留白充足。
排版：上方标题区，中部左右双栏 Lookbook 对照，左侧少而孤立的单品，右侧可互搭的衣橱组合，底部一句总结。
内容层级：大标题「核心不是越少越好」，副标题/提示「胶囊衣橱真正解决的是搭配问题」，要点包括「误解：胶囊衣橱 = 少买」「正确：单品更容易互相搭配」「关键：组合效率」，总结「减少的是决策成本」。
视觉锚点：左右对比衣物平铺图，右侧通过细线标签表现单品之间可组合。
色彩：奶油白背景，深巧克力文字，焦糖棕线条，少量橄榄绿和酒红点缀，低饱和。
避免：消费克制说教、科技知识卡风格、复杂架构图、密集小字、不可读中文。

### 内容页2/3

[3:4, 1K] 生成一张小红书图文卡片，页型是「框架页」，主题是「胶囊衣橱三类单品」。
画面气质：法式电商杂志风，精品电商分类网格，暖色棚拍，轻微胶片颗粒，粗窄体标题，优雅留白。
排版：上方强标题区，中部三格规则图片网格，每格一类单品和一个短说明，底部细线总结。
内容层级：大标题「好用衣橱的 3 类单品」，副标题/提示「先搭系统，再买单品」，要点包括「基础底色：决定可组合程度」「质感外层：让搭配更完整」「少量记忆点：形成个人风格」，总结「少量重点，比盲目追新更有用」。
视觉锚点：三格精品分类网格，分别展示白衬衫/牛仔裤、风衣/开衫、酒红包/丝巾等低饱和静物。
色彩：奶油白、焦糖棕、深巧克力文字，少量酒红和橄榄绿作为记忆点，柔和光影。
避免：电商货架感、过多商品、价格促销、黑金奢华、霓虹科技、卡通插画。

### 内容页3/3

[3:4, 1K] 生成一张小红书图文卡片，页型是「清单页」，主题是「胶囊衣橱购买前检查」。
画面气质：法式电商杂志风，奶油白纸感，焦糖棕细线，优雅购物清单，真实感布料和配饰静物，留白充足。
排版：上方标题区，中部三条优雅清单，每条配小静物或细线标签，底部结论，边距稳定，手机端可读。
内容层级：大标题「买之前先问 3 个问题」，副标题/提示「让新单品回到真实衣橱里」，要点包括「能和已有 3 件单品搭配吗？」「适合我的真实生活场景吗？」「颜色和材质能融入现有衣橱吗？」，总结「值得买 = 能让更多搭配成立」。
视觉锚点：优雅衣橱检查卡，旁边有折叠衬衫、丝巾、皮革腰带等低饱和静物。
色彩：奶油白背景，深巧克力文字，焦糖棕细线，酒红或橄榄绿小点缀，柔和棚拍阴影。
避免：价格、品牌、促销信息、密集商品目录、科技风、随机英文假字、不可读中文。
